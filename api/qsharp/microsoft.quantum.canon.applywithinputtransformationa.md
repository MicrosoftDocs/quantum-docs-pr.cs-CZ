Obor názvů: [Microsoft.. Canon](xref:Microsoft.Quantum.Canon)

```qsharp
operation ApplyWithInputTransformationA<'T, 'U> (fn : ('U -> 'T), op : ('T => Unit is Adj), input : 'U) : Unit is Adj
```
