Vypočítá logaritmus o základu 2 čísla.

```qsharp
function Lg (input : Double) : Double
```


## <a name="input"></a>Vstup

### <a name="input--double"></a>vstup: [Double](xref:microsoft.quantum.lang-ref.double)
