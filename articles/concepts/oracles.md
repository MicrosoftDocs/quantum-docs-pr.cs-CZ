# <a name="quantum-oracles"></a><span data-ttu-id="40f79-180">Nejenom Oracle</span><span class="sxs-lookup"><span data-stu-id="40f79-180">Quantum Oracles</span></span>

<span data-ttu-id="40f79-181">Oracle $ O $ je operace "černého pole", která se používá jako vstup pro jiný algoritmus.</span><span class="sxs-lookup"><span data-stu-id="40f79-181">An oracle $O$ is a "black box" operation that is used as input to another algorithm.</span></span>
<span data-ttu-id="40f79-182">Tyto operace se často definují pomocí klasické funkce $ f: \\ { 0, 1 \\ } ^ n \to \\ { 0, 1 \\ } ^ m, $ která přebírá $ n $ -bit binárního vstupu a vytváří $ $ binární výstup m-bit.</span><span class="sxs-lookup"><span data-stu-id="40f79-182">Often, such operations are defined using a classical function $f : \\{0, 1\\}^n \to \\{0, 1\\}^m$ which takes an $n$-bit binary input and produces an $m$-bit binary output.</span></span>
<span data-ttu-id="40f79-183">Provedete to tak, že zaberete konkrétní binární vstup $ x = (X_ { 0 } , x_ { 1 } , \dots X_ { n-1 } ) $ .</span><span class="sxs-lookup"><span data-stu-id="40f79-183">To do so, consider a particular binary input $x = (x_{0}, x_{1}, \dots, x_{n-1})$.</span></span>
<span data-ttu-id="40f79-184">Qubit je možné označit jako $ \ket { \vec { x } } = \ket { X_ { 0 } } \otimes \ket { X_ { 1 } } \otimes \cdots \otimes \ket { X_ { n-1 } } $ .</span><span class="sxs-lookup"><span data-stu-id="40f79-184">We can label qubit states as $\ket{\vec{x}} = \ket{x_{0}} \otimes \ket{x_{1}} \otimes \cdots \otimes \ket{x_{n-1}}$.</span></span>

<span data-ttu-id="40f79-185">Můžeme se nejdřív pokusit o definování $ o $ , aby bylo $ o \ket { × } = \ket { f (x) } $ , ale to má několik problémů.</span><span class="sxs-lookup"><span data-stu-id="40f79-185">We may first attempt to define $O$ so that $O\ket{x} = \ket{f(x)}$, but this has a couple problems.</span></span>
<span data-ttu-id="40f79-186">V prvním $ případě $ může mít jazyk f jinou velikost vstupu a výstupu ( $ n \ne m $ ), takže při použití $ $ hodnoty O by došlo ke změně počtu qubits v registru.</span><span class="sxs-lookup"><span data-stu-id="40f79-186">First, $f$ may have a different size of input and output ($n \ne m$), such that applying $O$ would change the number of qubits in the register.</span></span>
<span data-ttu-id="40f79-187">Druhý, i když $ n = m $ , funkce nemusí být inverzi: Pokud $ f (x) = f (y) $ pro určitý $ x \ne y $ , pak $ o \ket { x } = o y, \ket { } $ ale $ o ^ \dagger o \ket { x } \ne o ^ \dagger o \ket { y } $ .</span><span class="sxs-lookup"><span data-stu-id="40f79-187">Second, even if $n = m$, the function may not be invertible: if $f(x) = f(y)$ for some $x \ne y$, then $O\ket{x} = O\ket{y}$ but $O^\dagger O\ket{x} \ne O^\dagger O\ket{y}$.</span></span>

$$
\begin{align}
